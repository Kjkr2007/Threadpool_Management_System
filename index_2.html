03

System Flow

1
Pool Initialization
ThreadPool spawns N worker threads at startup. Each thread enters a blocking wait on the shared task queue.
pool = ThreadPool(pool_size=4)
pool.start() # spawns 4 WorkerThreads
2
Task Submission
Client calls pool.submit(fn, args). Task is wrapped in a Task object with timestamps and pushed to the thread-safe queue.
task = pool.submit(my_fn, args=(42,))
stats["tasks_submitted"] += 1
3
Task Pickup & Execution
An idle worker wakes via condition variable signal, dequeues the task, records started_at, and executes the function.
task = queue.get(timeout=0.5)
task.started_at = time.time()
task.result = task.fn(*task.args)
4
Completion & Reuse
Worker records completed_at, calls task_done(), appends to completed_tasks, and returns to idle — ready for the next task.
task.completed_at = time.time()
queue.task_done()
self.state = "idle"
5
Graceful Shutdown
pool.shutdown() sends one poison-pill (None) per active worker. Each worker finishes its current task, then exits cleanly.
for _ in active_workers:
queue.put(None) # poison pill
for w in workers: w.join()
